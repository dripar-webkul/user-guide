# Magic AI

To add Magic AI go to Admin Panel and go to **Configuration >> Magic AI** as shown in the below image.

![Magic AI Configuration](/assets/1.0/images/configuration/AiConfiguration.png)

In the configuration panel, you can set:

- **System Prompt**: Define the global behavior of the AI assistant.
- **Max Tokens**: Limit the response length to manage costs and output quality.
- **AI Providers**: Configure API keys for OpenAI, Gemini, Ollama, etc.

And, then click on **Save Configuration** button.
